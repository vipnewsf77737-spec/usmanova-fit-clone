# UsmanovaFit landing clone

Мини-копия трех экранов лендинга `https://usmanovafit.gymteam.ru/mainpage` для тестового задания.

## Что сделано

- Первый экран с CTA.
- Блок программ с рабочими кнопками `Подробнее`.
- Блок подбора программы с формой.
- FAQ-аккордеон и cookie-плашка.
- Адаптив для телефона и десктопа.

## Как открыть локально

Откройте `index.html` в браузере или запустите простой статический сервер:

```bash
python -m http.server 8080
```

## Деплой на GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в него файлы `index.html`, `styles.css`, `script.js`, `README.md`.
3. Откройте `Settings -> Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохраните. Через пару минут ссылка появится в этом же разделе.


Инструмент: Codex, HTML/CSS/JavaScript.
